# Client Onboarding Checklist

## New Client Setup Process

### Phase 1: Initial Setup (Day 1)
- [ ] **Client Information Collected**
  - Client name and business type
  - Owner name and contact information
  - Business address
  - Website and email
  - Licensed states
  - Target industries

- [ ] **Airtable Entry Created**
  - Add client to Airtable database
  - Set status to "Onboarding"
  - Assign appointment setter (Brendan/Jacques)
  - Enter all client details

- [ ] **GoHighLevel Phone Number Assignment**
  - Assign dedicated phone number
  - Configure call routing
  - Set up voicemail
  - Add to GHL contacts

### Phase 2: Team Setup (Days 2-3)
- [ ] **Team Members Added**
  - Gather all team member names
  - Collect contact information
  - Note roles and responsibilities
  - Add to Airtable team section

- [ ] **Communication Preferences**
  - Identify decision makers
  - Note best contact times
  - Timezone confirmation
  - Preferred communication method

### Phase 3: Business Details (Days 3-5)
- [ ] **Target Market Information**
  - Target industries identified
  - Priority states confirmed
  - Revenue goals documented
  - Close rate expectations set

- [ ] **Carrier Information**
  - Top carriers listed
  - Specialty markets noted
  - Special requirements documented

- [ ] **Account Details**
  - Current monthly premium
  - Goal monthly premium
  - Special notes/restrictions

### Phase 4: Go Live (Days 5-7)
- [ ] **Cheat Sheet Updated**
  - All information added to cheat sheet
  - Status changed to "Active"
  - Team notified of new client

- [ ] **First Appointment Scheduled**
  - Initial check-in meeting set
  - Setter assigned and ready
  - Goals and expectations aligned

- [ ] **Documentation Complete**
  - All fields in Airtable complete
  - Cheat sheet displaying correctly
  - Phone numbers tested
  - Team has access

### Common Onboarding Issues
- **Missing team member contacts** - Follow up with client
- **Unclear target industries** - Schedule discovery call
- **No GoHighLevel number** - Contact admin for assignment
- **Incomplete state licenses** - Verify with client records

## Onboarding Timeline
- **Day 1**: Initial data collection
- **Days 2-3**: Team setup and communication preferences
- **Days 3-5**: Business details and target markets
- **Days 5-7**: Go live and first appointment
- **Week 2**: First check-in and optimization
